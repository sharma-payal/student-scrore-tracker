# Student Score Tracker (SQL Practice Project)

This project is a hands-on SQL learning repository built using Git and GitHub workflows.

## What I practiced here:
- SQL Basics (SELECT, WHERE, ORDER BY, LIMIT)
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY and HAVING
- Table relationships
- SQL JOINs (INNER JOIN, LEFT JOIN)
- Git branching workflow
- Pull Requests on GitHub

## Tech Used:
- SQL
- Git
- GitHub
- Terminal

## Goal:
To build strong SQL fundamentals for Data Analyst / Data Engineer roles using real practice and version control.
